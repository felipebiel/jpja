# JPJÁ
<p>
Sistema para aprendizado em linguás focado no japonês.
</p>
<p>
Desenvolvido no segundo semestre na matéria de projeto 1, o sistema e site JPjá foi desenvolvido em Laravel 5.2, com bootstrap 3, CSS e JQuery.
O mesmo consistia em gerar aprendizado aos seus usuários na linguagem em japones, semelhante ao Duolingo
</p>
